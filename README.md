# Adaptiv Landing Page

A modern, responsive landing page for Adaptiv - AI-powered personalized snack delivery service.

## Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Modern, clean UI with smooth animations
- ✅ Waitlist signup form with validation
- ✅ SEO-friendly HTML structure
- ✅ Ready to deploy

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and responsive design
- `script.js` - Form handling and interactions

## How to Use

1. **Local Testing**: Simply open `index.html` in your web browser

2. **Deploy to Web**: 
   - Upload all files to any web hosting service (GitHub Pages, Netlify, Vercel, etc.)
   - For GitHub Pages: Push to a repository and enable GitHub Pages in settings
   - For Netlify: Drag and drop the folder to Netlify
   - For Vercel: Use Vercel CLI or connect your GitHub repo

## Customization

To integrate the waitlist form with a real backend:
1. Replace the form submission logic in `script.js` (around line 25)
2. Add your API endpoint or third-party service integration
3. Popular options: Google Sheets API, Mailchimp, ConvertKit, or your own backend

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
